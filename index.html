响应式

1.下载地址：https://browsersync.io 2.点击get started按钮安装，复制提示的地址，前面加为cnpm install -g browser-sync，使用快捷键window+r,输入cmd运行进入--黏贴地址回车安装 3.安装完成后，在文件所在文件夹按住shif键右键选择在此处打开命令窗口---也可使用快捷键window+r,输入cmd运行进入；输入cd 加上本地文件目录回车，再输入browser-sync start --server --files="*" 自动跳入命名为index.html的文件（http://localhost:3000/） 4.手机浏览器中输入IP地址例如192.168.1.7加上:3001则就进入了Browsersync 工具；pc端网页地址也许使用IP地址访问（同一网址），否则手机页面无法正常打开； 5.改变index文件内容，可在移动端时时更新 6. pc端的touch事件和移动端的touch事件存在兼容性问题， 所以无法同步。
